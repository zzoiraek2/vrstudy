## Response
| 구분 | Element | 한글명 | Type | Required | Length | Description |
| --- | --- | --- | --- | --- | --- | --- |
| Header | api-id | TR명 | String | Y | 10 | 7자리 TR코드, ex) ka00001 |
|  | cont-yn | 연속조회여부 | String | N | 1 | 다음 데이터가 있을시 Y값 전달 |
|  | next-key | 연속조회키 | String | N | 50 | 다음 데이터가 있을시 다음 키값 전달 |
| Body | result_list | 결과리스트 |  | N |  |  |
|  | - cur_prc | 현재가, 종가 |  | N |  | 단위: USD, 부호 포함 소수점 넷째 자리까지 포맷된 숫자 |
|  | - pred_pre | 전일대비 |  | N |  | 단위: USD, 부호 포함 소수점 넷째 자리까지 포맷된 숫자 |
|  | - flu_rt | 등락률 |  | N |  | 단위: %, 부호 포함 소수점 둘째 자리까지 포맷된 백분율 |
|  | - trde_qty | 거래량 |  | N |  | 단위: 1주, 부호가 포함된 숫자 |
|  | - cntr_tm | 체결시간 |  | N |  | HHmmss |
|  | - pred_pre_sig | 전일대비기호 |  | N |  | 1: 상한가, 2:상승, 3:보합, 4:하한가, 5:하락 |
